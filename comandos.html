Main

significa "contenido principal". Se utiliza para representar el contenido principal de un documento o aplicación.

Section

Se utiliza para representar una sección independiente de un documento o aplicación. Una sección puede contener cualquier contenido, como texto, imágenes, videos, etc.

Article

se utiliza para representar una composición autocontenida en un documento, una página, una aplicación o un sitio, puede ser un artículo de blog, una entrada de wiki, un comentario de un foro, o una pieza de contenido multimedia se pueden representar mediante un elemento

"contiene un título, un párrafo y una imagen. Este contenido es un artículo independiente que puede ser distribuido o reutilizado de manera independiente."

Pre

significa "preformateado". Se utiliza para representar texto preformateado, que es texto que se muestra exactamente como se escribe en el archivo HTML.

"contiene un párrafo de texto preformateado. El texto se muestra en una fuente fija, y los espacios y los saltos de línea se conservan."

Header

se utiliza para representar el encabezado de un documento o sección. El encabezado generalmente contiene información importante, como el título de la página, el logotipo de la empresa, o la navegación principal.

"contiene un título, un logotipo, y una navegación principal. Este contenido es el encabezado de la página web."

Footer

se utiliza para representar el pie de página de un documento o sección. El pie de página generalmente contiene información de contacto, derechos de autor, o enlaces a documentos relacionados.

Figure

representa contenido autocontenido, como imágenes, ilustraciones, diagramas, fragmentos de código, etc. El elemento figure puede contener cualquier contenido, pero generalmente se usa para contener contenido que puede ser movido o eliminado sin afectar el flujo principal del documento.

contiene una imagen de una manzana. La imagen está rodeada de un borde y tiene una leyenda que explica la imagen.

Figcaption

representa una leyenda o título para una figura. Se utiliza en combinación con el elemento figure para agrupar una imagen u otro contenido multimedia con su título o leyenda correspondiente.

Aside

representa una sección de contenido que está indirectamente relacionado con el contenido principal de una página web. Este contenido a menudo se muestra como una barra lateral o un bloque de contenido que se puede quitar sin afectar la comprensión del contenido principal.

"contiene una barra lateral con el título "Esta es una barra lateral". El contenido de la barra lateral es independiente del contenido principal de la página."

Dfn

representa un término que se define en el contexto de una frase o oración. El elemento dfn se utiliza para ayudar a los lectores a entender el significado de un término técnico o especializado.

U

Este elemento representa un texto subrayado. El texto dentro del elemento u se muestra con un subrayado.

Small

representa un texto de menor tamaño que el texto circundante. El texto dentro del elemento small se muestra en una fuente más pequeña en la mayoría de los navegadores web.

Div

representa una división o una sección en un documento HTML. Es un elemento de contenedor genérico que se puede utilizar para agrupar cualquier tipo de contenido.

Code

representa un fragmento de código. El contenido dentro del elemento code se muestra en una fuente monoespaciada, que es la fuente que se utiliza generalmente para representar código.

Var

representa el nombre de una variable en una expresión matemática o en un contexto de programación. El contenido dentro del elemento var se muestra en una fuente itálica, aunque ese comportamiento depende del navegador.

"se utiliza para representar las variables l, w y h. El contenido dentro del elemento var se muestra en una fuente itálica."

Samp

representa una muestra de salida de un programa de computadora. El contenido dentro del elemento samp se muestra en una fuente monoespaciada, que es la fuente que se utiliza generalmente para representar código.

"Ayuda a distinguir visualmente la salida del programa de resto del contenido"

Kbd

representa un texto que representa una entrada de teclado. El contenido dentro del elemento kbd se muestra en una fuente monoespaciada, que es la fuente que se utiliza generalmente para representar código.

"En una página web, el elemento kbd se puede utilizar para mostrar atajos de teclado, comandos de consola o códigos de programación."

Mark

El elemento mark se puede utilizar para resaltar texto que es importante, relevante o que el usuario debe prestar atención. Por ejemplo, se puede utilizar para resaltar palabras clave en un documento, resultados de búsqueda o información importante en un texto.

Time

representa un momento específico en el tiempo. Puede incluir el atributo datetime para convertir las fechas en un formato interno legible por un ordenador, permitiendo mejores resultados en los motores de búsqueda o características personalizadas como recordatorios.

El elemento time se puede utilizar para representar:
* Una fecha, como el cumpleaños de una persona.
* Una hora, como la hora de salida de un tren.
* Una fecha y hora, como la fecha y hora de un evento.

ins

representa un fragmento de texto que ha sido añadido al documento. Este elemento se puede utilizar para marcar texto que ha sido modificado o corregido.

"subrayado"

Del

representa un fragmento de texto que ha sido eliminado del documento. Este elemento se puede utilizar para marcar texto que ha sido modificado o corregido.

"Texto tachado"
Hola

Data

se utiliza para almacenar información adicional sobre un elemento HTML. Esta información puede ser utilizada por scripts para interactuar con el elemento HTML.

sub

se utiliza para representar texto que se debe mostrar en subíndice. El texto dentro del elemento sub se muestra en una línea inferior a la línea principal del texto.

"La fórmula para calcular el área de un círculo es: πr²"

Sup

se utiliza para representar texto que se debe mostrar en superíndice. El texto dentro del elemento sup se muestra en una línea superior a la línea principal del texto.

"El año de nacimiento de Albert Einstein es 1879."

wbr

representa una oportunidad de salto de línea, un lugar en el texto donde el navegador puede opcionalmente romper una línea, aunque sus reglas de salto de línea de otra manera no crearían una ruptura en esa posición.

Br

representa un salto de línea en el texto. El texto después del elemento br comienza en el principio de la línea siguiente.

"Este es un párrafo con un
salto de línea."

P

significa "párrafo". Se utiliza para representar un párrafo de texto.

Img

Se utiliza para incrustar una imagen en una página web. Las imágenes se pueden utilizar para mejorar la estética de una página web, para proporcionar información adicional o para atraer la atención del usuario.

Svg

Se trata de un lenguaje de marcado basado en XML que se utiliza para crear gráficos vectoriales. Los gráficos vectoriales son imágenes que se crean a partir de objetos matemáticos, como líneas, círculos y polígonos. Esto significa que los gráficos vectoriales se pueden escalar sin perder calidad.

Atributos de SVG

width: Especifica el ancho del SVG.
height: Especifica la altura del SVG.
viewBox: Especifica el área de visualización del SVG.
preserveAspectRatio: Especifica cómo se debe mantener la relación de aspecto del SVG al escalarlo.
class: Una clase CSS para aplicar estilos al SVG.
id: Un identificador único para el SVG.

Caption

se utiliza para especificar la leyenda (o título) de una tabla. El elemento caption debe ser el primer hijo de su elemento table padre.


Th

significa "encabezado". Se utiliza para definir una celda como un encabezado de un grupo de celdas en una tabla. Las celdas de encabezado se suelen utilizar para mostrar información importante sobre los datos de la tabla.


Td

significa "datos". Se utiliza para definir una celda como una celda de datos en una tabla. Las celdas de datos suelen utilizarse para mostrar información de la tabla.

Tr

significa "fila". Se utiliza para definir una fila de celdas en una tabla. Las filas se utilizan para organizar los datos de la tabla en columnas.

colgroup

significa "grupo de columnas". Se utiliza para definir un grupo de columnas dentro de una tabla. Los grupos de columnas se pueden utilizar para aplicar estilos a un conjunto de columnas, en lugar de repetir los estilos para cada celda, para cada fila.

Atributos del elemento Colgroup


* width: Define la anchura de las columnas del grupo.
* span: Define el número de columnas que se agrupan. Un número entero mayor que cero. Por defecto: 1.

Col

significa "columna". Se utiliza para definir una columna dentro de una tabla. Los elementos col se pueden utilizar para aplicar estilos a una columna o a un grupo de columnas, en lugar de repetir los estilos para cada celda, para cada fila.
